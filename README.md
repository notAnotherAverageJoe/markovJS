# 📜 Markov Text Generator

Welcome to the Markov Text Generator! This project uses a Markov chain to generate random text based on an input file or URL.

## 🚀 Features

- **Generate Text from Files**: Read text from a file and generate random text.
- **Generate Text from URLs**: Fetch text from a URL and generate random text.
- **Markov Chain Implementation**: A simple Markov chain algorithm to produce the text.

## 🛠️ Installation

1. **Clone the Repository:**

    ```bash
    git clone https://github.com/yourusername/CLI_todos_python.git
    cd CLI_todos_python
    ```

2. **Install the Dependencies:**

    ```bash
    npm install
    ```
